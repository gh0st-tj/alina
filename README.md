# Reconnection Landing Page

A beautiful, modern landing page designed to help reconnect with someone special. Built with Next.js and Tailwind CSS.

## Features

- Modern, responsive design
- Beautiful photo slider
- Easily editable content
- Smooth animations and transitions
- Mobile-friendly layout

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone this repository
2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Customizing Content

All the text content is stored in `src/content/pageContent.js`. You can easily edit this file to update the text throughout the landing page.

## Adding Your Photos

1. Place your photos in the `public/images` directory
2. For the photo slider, update the image paths in `src/components/PhotoSlider.js`
3. For other images used throughout the site, update the image paths in their respective component files

## Customizing Colors

The color scheme can be customized in the `tailwind.config.js` file. The default colors are:

- Primary: #ff6b6b (a soft red)
- Secondary: #4ecdc4 (a teal color)
- Dark: #1a1a2e (a deep blue-black)
- Light: #f7f7f7 (an off-white)

## Deployment

This is a Next.js project, so it can be easily deployed to platforms like Vercel, Netlify, or any other hosting service that supports Node.js.
